object oriented programming

Object-oriented programming (OOP) is a programming paradigm that provides a set of features that enable developers to create software systems using objects that interact with each other.

OOP features include encapsulation, inheritance, polymorphism, abstraction, classes, objects, and methods. OOP provides a powerful set of features that help to create modular, reusable, and maintainable code.
